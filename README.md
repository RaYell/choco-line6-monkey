# This project is no longer maintained!

# Description
[Chocolatey](https://chocolatey.org) package for [Line 6 Monkey](http://line6.com/software).

# Installation
```posh
choco install Line6Monkey
```

# Uninstallation

```posh
choco uninstall Line6Monkey
```

Note that the uninstall command will only remove the Chocolatey package as Line6 Monkey does not provide method to be silently uninstalled.
